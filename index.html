Exercice 2.3

Programme CalculTTC

Var
nomArt: chaine de Caractères
prix, nombre, prixht: entier
tva, PrixTTC: réel

Début

Ecrire ("Entez le nom de l Article")

Lire(nomArt)

Ecrire ("Entez le nombre d Article achete ")

Lire(nombre)

Ecrire ("Entez le prix unitaire de l Article")

Lire(prix)

prixht ← prix * nombre

tva ← 0,16 * prixht

Ecrire("Le prix HT est :"; prixht)

Ecrire("Le prix TTC est :"; PrixTTC)

Fin.
